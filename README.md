# ESP32 FastAccelStepper Web Control

Тестовый проект управления шаговым/сервошаговым двигателем на **ESP32**  
с использованием библиотеки **FastAccelStepper** и встроенного **Web-интерфейса**.

Проект задумывался как песочница для:
- генерации STEP/DIR на ESP32,
- удалённого управления через браузер,

---

## Возможности

- STEP/DIR генерация на ESP32
- Плавный разгон и торможение (acceleration)
- Команды:
  - start / stop
  - установка частоты (f)
  - установка ускорения (acc)
  - ramp (плавный выход на частоту за заданное время)
  - направление (dir)
  - enable (en)
- Web-интерфейс:
  - управление из браузера
  - live-статусы
- WiFi-конфигурация через `.env` (не попадает в git)
